Create a layer from a KML file

We can create a new layer from a KML file using the button in Sextant's main menu bar and then select to Load KML layer as shown in the figure below.


We then have to fill in the following form to create our layer. The first field is the name of the layer. Layer names are unique in Sextant, so each layer must have a different name. In the URI field we can provide a URI of a KML file in case it is a web resource, or we can click on the Browse button to select a local KML file from our file system. If the file has temporal information, we can select the Is a temporal layer checkbox to create the associated events in the Timeline.
