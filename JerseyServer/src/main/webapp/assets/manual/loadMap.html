Loading a map

To load a map in Sextant if we know the mapid, apart from using the URI of the map, we can use the button in Sextant's main menu bar and then select Load from MapID as shown in the figure below.


We then have to fill the following form to load the map. The first field is the Map ID where we provide the unique identifier for the map. The next two fields are used to point the map registry in which the map is saved. In the URI field we provide the URI that is used to query the registry and in the Port field the port number in which the registry listens to.



The user can leave the Endpoint Information fields empty. In this case Sextant will use the default map registry that is provided in the server-configuration.properties under /Sextant/WEB-INF/classes/server as described in Save map manual page.

When we load a map all the layers are available in the Layers panel. The URL in our browser changes to match the URI of the map and the Map information panel is populated with the map metadata and can act as a legend, as shown in the figure below.
