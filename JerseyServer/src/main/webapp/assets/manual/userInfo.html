Add POIs and AOIs on the map

Users can add points of interest (POIS) and areas of interest (AOIs) on the map using the buttons at the upper left corner of the map. These features are added in the userInfo default layer. When we click on one of the button the cursor changes and we can draw a point/rectangle on the map and then fill the provided form with the information for the feature we created.
