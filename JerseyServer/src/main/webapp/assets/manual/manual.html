Screen Layout

The screen layout is divided into three parts:

  • The top action bar contains the application functions. It displays all actions as icons when the screen resolution is wide enough. Otherwise, these actions are provided as menu entries accessible by the virtual or hardware menu key

  • The main content view provides a map base layer where the contents of thematic maps are visualized into layers, and a Timeline to view the evolution of layers through time

  • The side panel provides several function buttons over the layers, the map information and the explore function
