Layer manipulation functions

Each layer in the Layers panel is accompanied by a set of buttons that allow certain functionalities over the layer as shown in the figure below.


The supported functions are the following:

  • Show or Hide: Available for all types
  • Zoom: Available for all types
  • Info: Available for all types
  • Update: Available for layers produced by queries
  • Global Styles: Available for KML, GML and layers produced by queries
  • Feature Styles: Available for KML, GML and layers produced by queries
  • Spatial Filter: Available for KML, GML and layers produced by queries
  • Move on top: Available for all types
  • Download: Available for all types except WMS and userInfo layer
  • Delete: Available for all types except WMS and userInfo layer
