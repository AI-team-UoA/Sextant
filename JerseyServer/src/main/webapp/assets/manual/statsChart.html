Produce charts using layers' features

We can create charts using the values of a layer's feature, using the Feature charts panel shown in the figure below


When we load a vector layer in Sextant, the Layer selection list is updated with the name of the layer. When we choose one of the available layers from the Layer drop-down list, the Attribute list is enabled and we can choose one of the numeric attributes of the selected layer. When the attribute is selected, a chart is produced as shown in the figure below


The produced charts are responsive and their features are connected to the features on the map, so when we click on a point in the chart, we view the selected feature on the map. We can also download the chart as an image to the download folder of the browser, using the button.
