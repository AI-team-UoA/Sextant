Working with proxy server

If Sextant is installed under a proxy server, in order to communicate with the application we can set the parameters of the proxy in the server-configuration.properties file under /Sextant/WEB-INF/classes/server. There are three parameters that should be set in the configuration file:

  • PROXY_STATUS = on/off
  • PROXY_NAME = URI of the Proxy server
  • PROXY_PORT = The port that the Proxy server listens to
