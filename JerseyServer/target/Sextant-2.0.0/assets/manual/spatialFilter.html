Filter the layer to a given location

We can apply a spatial filter on a layer by clicking the button of the layer in the Layers panel as shown in the picture below.


As a result we get a pop-up where we can select an area of interest, as shown in the figures below. In the first drop-down list we can choose a country. Then all the available regions of that country populate the next drop-down list. If we choose a region then the next drop-down list is populated with all the region units for this region. Finally if we select a region unit, the last drop-down list is populated with all the cities in this region unit. We can choose to stop at any point after the country selection to specify the desired location for the spatial filtering. When we click the Apply button, all the features of the layer tha are outside the boundaries of the specified location are styled with grey colors, while the rest are left unchanged. If we click the Clear button, any spatial filter on the layer is removed and all the features of the layer retrieve their original styles.
