Create a layer from a SPARQL Endpoint

We can create a new layer by posing a query to a SPARQL endpoint using the button in Sextant's main menu bar and then select to Pose a Query as shown in the figure below.


We then have to fill in the following form to create our layer. First we can select the type of the endpoint. We can choose between SPARQL endpoint and the Ordnance Survey SPARQL endpoint. If we choose the SPARQL endpoint option we need to provide the URI of the endpoint in the field URI and the port it listens to in the field Port:80 (by default the port is set to 80). If we choose the OS SPARQL endpoint these fields are filled automatically. The next field is the name of the layer. Layer names are unique in Sextant, so each layer must have a different name. In the Query field we provide the SPARQL query, the results of which will create our layer. To write our query we can use GeoSPARQL or stSPARQL if the endpoint is a spatiotemporal Strabon endpoint. If the query also returns temporal information, we can select the Is a temporal layer checkbox to create the assosiated events in the Timeline.
