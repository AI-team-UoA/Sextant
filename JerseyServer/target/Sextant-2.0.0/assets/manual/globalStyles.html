Change a layer's style

We can change a layer's style by clicking the button of the layer in the Layers panel as shown in the picture below.


As a result we get a pop-up where we can choose the following attributes, as show in figure below:

  • Stroke Color: The color of the exterior of the layer's polygons
  • Stroke Width: The width in pixels of the exterior of the layer's polygons
  • Fill Color: The color of the interior of the layer's polygons
  • Icon URI: An image to represent the points of a layer
  • Icon size: The width of the image that represents the points, in pixels

In the case of temporal layers, Stroke Color is also used in the Timeline to show period events for the layer and Icon URI to show a timestamp events. Empty fields in the form are given default values automatically.



When we click on the Stroke Color or the Fill Color buttons, we get the color selector shown below, where we can choose a color and click the OK button in the lower right corner to select it.
