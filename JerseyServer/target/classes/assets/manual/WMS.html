Create a layer from a WMS service

We can create a new layer from a WMS service using the button in Sextant's main menu bar and then select to Load WMS layer as shown in the figure below.


We then have to fill in the following form to create our layer. The first field is the name of the layer. Layer names are unique in Sextant, so each layer must have a different name. In the WMS server URI field we provide a URI of the desired WMS service. Using the two drop-down lists that follow we select the server type [geoserver, mapserver, qgis] and the version [1.1.x, 1.3.x]. If the name of WMS layer is know to us, we can use it to fill in the WMS layer field and create the layer using it's default style. In case we want to view the available layers in the WMS service we can click the Get Capabilities button. Doing that will populate the drop-down list under the WMS layer field with the names of the layers the WMS service can provide. We can then choose one of these and the WMS layer field will be updated automatically so that we can create the layer. When we choose a layer from the drop-down list, the styles list will be updated with all the available styles for the layer. We can choose a different style than the default style to use the specified style for rendering.
