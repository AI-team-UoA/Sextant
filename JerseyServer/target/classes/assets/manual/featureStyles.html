Change a layer's style according to a feature's values

We can change a layer's style according to the values of a specific feature by clicking the button of the layer in the Layers panel as shown in the picture below.


As a result we get a pop-up where we can select the desired feature, the data type for the comparisons (numeric or string) and adjust the interval's boundaries, as shown below. To add more intervals we can click the button next to the interval's color. We can change the default color for each interval by clicking on the color button, choose a color in the color-picker that appears and click the OK button in the bottom right corner of the color-picker. When the styling takes place, the Color Panel appears in the left side of the UI. We can show the Color Panel at any point by using the button in Sextant's main menu bar and then select Show Color Panel from the basic functions.
